# Calendar++

**Calendar++** est une bibliothèque C++ header-only pour manipuler, convertir et afficher des dates dans **16 calendriers** du monde entier. Zéro dépendance, un seul fichier `calendar.h`.

## 🌍 Calendriers supportés
Grégorien, Islamique, Hébraïque, Chinois (zodiaque), Maya (compte long), Persan, Éthiopien, Copte, Hindou, Bouddhiste, Zoroastrien, Arménien, Républicain, Bahá'í, Sikh, Julien.

## 🗣️ Langues
Français, Anglais, Arabe, Hébreu, Chinois, Persan.

## ✨ Fonctionnalités
- Conversions bidirectionnelles
- Opérations arithmétiques (jours/mois/années)
- Affichage de calendriers (mensuel, annuel)
- Validation et normalisation
- Fêtes et jours spéciaux
- Formatage personnalisé
- Comparaison complète
